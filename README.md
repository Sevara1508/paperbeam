# PaperBeam 🚀

A documentation site for the fictional web technology **PaperBeam**, built using [VitePress](https://vitepress.dev/).
Created as part of Lab 07-08 for my Web Development course at Ontario Tech University.

## 📖 About

PaperBeam is a fictional content publishing platform designed to help teams organize,
write, and share structured content efficiently. This site showcases its features,
API reference, and learning resources.

## 📄 Pages

- **Products** — Overview of PaperBeam's plans and core features
- **API** — Technical reference and code examples
- **Contact** — How to get in touch with the PaperBeam team
- **Learning/Docs** — Getting started guide and setup instructions
- **Learning/Tutorials** — Step-by-step walkthroughs
- **Learning/Cheatsheets** — Quick reference tables and markdown syntax

## 🛠️ Built With

- [VitePress](https://vitepress.dev/) — Static site generator
- Markdown — All page content
- Custom CSS — Brand colours and button styling

## 🚀 Running Locally

Clone the repo and install dependencies:

```bash
git clone https://github.com/yourusername/paperbeam.git
cd paperbeam
npm install
```

Start the dev server:

```bash
npm run docs:dev
```

Open your browser at `http://localhost:5173`

## 📦 Building for Production

```bash
npm run docs:build
```
